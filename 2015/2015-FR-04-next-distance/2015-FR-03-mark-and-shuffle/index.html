Marquage d'oiseaux

Afin d'étudier un groupe d'oiseaux migrateurs, vous souhaitez dessiner un motif sur une patte de chaque oiseau afin de pouvoir les reconnaître lorsqu'ils repasseront près de chez vous.

Choisissez un motif pour chaque oiseau en cliquant sur les cases sur les pattes. Puis cliquez sur le bouton pour les laisser s'envoler. Ils se mélangeront et il vous faudra les replacer dans l'ordre.
