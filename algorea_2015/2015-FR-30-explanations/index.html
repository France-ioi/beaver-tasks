Explications

Bienvenue ! Chacune des 9 questions contient trois versions du même problème, accessibles en cliquant sur les onglets en haut de la page. Ainsi, ceci est la version facilemoyennedifficile de cette question.

Votre score pour chaque question est le meilleur de vos scores parmi les trois versions. Les questions les plus difficiles rapportent plus de points.

Si vous trouvez que la version facile d'une question est très simple, vous pouvez résoudre directement la version moyenne ou même difficile pour gagner du temps.

Si par contre vous bloquez sur les versions moyennes ou difficiles de certaines questions, résolvez d'abord les versions faciles de toutes les questions. Résoudre toutes les versions faciles donne un peu plus de la moitié des points.

Vérifions que vous avez compris :

Résoudre la version difficile suffit pour avoir tous les points d'une question.
Résoudre la version facile de toutes les questions donne plus de la moitié des points.
On peut avoir tous les points d'une question sans résoudre la version moyenne.
